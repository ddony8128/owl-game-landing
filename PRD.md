# 부엉이게임 홍보 랜딩페이지 PRD

## 목적
1. **부엉이게임 시즌1(3/14), 시즌0(3/28) 참가자 모집**
2. **크몽 외주 포트폴리오용 샘플** — "행사/체험 홍보 + 예약 페이지 제작" 상품의 실제 사례

## 타겟 유저
- 두뇌게임/보드게임/지니어스에 관심 있는 20~30대
- 블로그/SNS를 통해 유입되는 잠재 참가자

## 기술 스택
- Next.js 15 (App Router)
- Tailwind CSS
- Framer Motion (애니메이션)
- Vercel 배포
- Google Forms 연동 (예약)

## 페이지 구성

### 1. 히어로 섹션
- 메인 카피: "모두가 주인공이 되는 두뇌 서바이벌"
- 서브 카피: "부엉이게임"
- 다크/미스터리 톤 배경 (부엉이게임 세계관 반영)
- CTA 버튼: "참가 신청하기" → 예약 섹션으로 스크롤

### 2. 게임 소개 섹션
- 한 줄 설명: "이 게임에 엑스트라는 없습니다."
- 3가지 키워드 카드:
  - 독창적인 두뇌게임 — 매 시즌 새로운 오리지널 게임
  - 실시간 웹 시스템 — 모바일로 참여하는 라이브 게임플레이
  - 당신이 주인공 — 전략, 협상, 배신. 당신의 선택이 결과를 바꿉니다.

### 3. 지난 시즌 하이라이트
- 실적 숫자: "2회 운영 / 8인 동시접속 / 5시간 풀 게임"
- 플레이어 후기 인용 (블로그에서 발췌)
- 게임 스크린샷 or 분위기 이미지
